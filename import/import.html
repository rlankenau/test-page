Introduction

Making the import process easier

Before you move on to importing your institution's data, please follow through the steps under Setting up your database to set your computer up to access the databse. This step needs to be done on any new computers that will be importing data to the database.

One of the goals of this project was to make it easy to update the data in the Access database, so users weren't spending a lot of time importing new data into their databases. We also wanted it to be easy for different institutions to adapt to their needs. To this end, there are a few data import programs included in the 'src' directory. I'll describe a bit about how they work in the following sections, but you should just be able to run them and follow along with the instructions to import new data to your database.

The programs are written in the Python programming language. If you don't yet have Python installed on your computer, take a look at Appendix A for details on how to install and run the software.

The section entitled Data in existing databases covers how to import data from existing databases, such as patron data stored in Voyager.

The final section, EZproxy Data, goes over how we can import data directly from text files.
