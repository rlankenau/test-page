Troubleshooting

This appendix contains information about errors that may occur while you're working with the software. If something isn't behaving as you expect, take note of any error messages reported, and check this appendix.

Error: ('IM014', '[IM014] [Microsoft][ODBC Driver Manager] The specified DSN contains an architecture mismatch between the Driver and Application')

This error may occur if you have installed a 32-bit version of Python, but set up a 64-bit ODBC DSN. Please check Appendix A for information about installing Python, and Setting up your database for information about DSNs.

If it turns out you installed a 32-bit Python, please install the 64-bit version, and then re-install the pypodbc package.
